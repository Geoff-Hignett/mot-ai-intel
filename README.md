# mot-ai-intel

AI-powered MOT risk analysis platform using React, .NET, and OpenAI. Users can analyse vehicle history and receive personalised maintenance insights.

## Architecture

- Frontend → React + TypeScript
- Backend → .NET 8 API
- AI Layer → OpenAI integration
